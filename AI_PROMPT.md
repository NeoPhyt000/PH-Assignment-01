# AI Prompt Documentation

## Project

**DevConf 2026 - Developer Conference Landing Page**

---

## AI Tools Used

- ChatGPT (GPT-5.5)
- Figma Make

---

## Objective

The project included an AI Challenge where the placeholder section ("Something Missing?") had to be replaced with a brand-new section relevant to the DevConf 2026 website.

My objective was to use AI to brainstorm suitable section ideas that matched the overall theme and visual style of the website. After selecting the most appropriate ideas, I used Figma Make to create a UI design for those sections. Finally, I implemented the selected design manually using HTML and CSS while ensuring it remained responsive and visually consistent with the rest of the landing page.

---

## Prompt Used in ChatGPT

```text
This is the ui of a website that i'm currently building. Basically this is based on a developer conference related website. With all the existing features i need a section for this ui where it is written as somethings misssing. it should be relevant to the website theme. give me some idea and design
```

---

## AI Response Summary

ChatGPT suggested several sections that would fit a developer conference website, including:

- Why Attend DevConf
- Conference Schedule
- Event Statistics
- Sponsors
- Testimonials
- FAQ
- Venue
- Technologies You'll Explore
- Newsletter
- Networking Timeline

After reviewing the suggestions, I selected the following sections because they best matched the existing website layout and theme:

- **What Attendees Say (Testimonials)**
- **Technologies You'll Explore**

---

## Prompt Used in Figma Make

```text
Design two modern website sections for a developer conference landing page named DevConf 2026.

The first section should be a "What Attendees Say" testimonial section with three clean testimonial cards, 5-star ratings, attendee names, job titles, and company names.

The second section should be titled "Technologies You'll Explore" and display modern pill-shaped technology badges including React, Next.js, Node.js, Docker, Kubernetes, AWS, Python, AI, Rust, Go, Flutter, and Cloud.

Use a clean, minimal design with rounded corners, subtle shadows, modern typography, and spacing that matches a professional developer conference website. Keep the design responsive and visually consistent with the rest of the landing page.
```

---

## Implementation

The UI generated by Figma Make was used as a visual reference. I then manually recreated the design using semantic HTML and CSS, adjusting spacing, typography, colors, and responsiveness to match the overall design of the DevConf 2026 landing page.

The Figma Make generated UI and screenshots are included in the **assets/** folder.

---

## AI Contribution

AI was used to:

- Generate relevant section ideas.
